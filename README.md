Currency Converter

This is a simple Currency Converter created in Google Colab using LangChain and Tool Calling.
The project takes an amount and converts it from one currency to another by using a tool that fetches the latest exchange rate.

Features

Convert any amount between two currencies

Uses LangChain to handle the conversation

Uses tool calling to get real-time currency rates

How to Use

Open the Colab notebook

Run all cells

Enter the amount and choose the currencies

The converted value will be displayed

Requirements

Python

LangChain

Any API/tool used for exchange rates

About

This project is a simple demo to understand how LangChain interacts with tools to perform tasks like currency conversion.
